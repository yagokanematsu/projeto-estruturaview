# Simulador de Fila Circular

Este site tem como objetivo **simular o funcionamento da estrutura de dados Fila Circular**, de forma visual e interativa, inspirado em plataformas educacionais como o *DebugandoED*.

O sistema representa exclusivamente a Fila Circular, abordando conceitos como:

- Estrutura sequencial com reaproveitamento de espaço
- Controle de início e fim da fila
- Condições de fila cheia e fila vazia
- Inserção (enqueue)
- Remoção (dequeue)

Cada operação realizada pelo usuário reflete diretamente na estrutura da fila, facilitando a visualização do comportamento circular.

## Objetivo educacional

O site foi desenvolvido com foco didático, sendo indicado para estudantes que estão iniciando o estudo de Estruturas de Dados. A visualização ajuda a transformar conceitos abstratos em algo mais concreto e compreensível.
